## Getting Started 
Clone the repository to your local machine.
```bash
git clone https://github.com/niasenago/PSI-project
cd PSI-project
```
Start the application
```bash
 dotnet run
 ```

 Open your web browser and navigate to http://localhost:7200 to access InfoSync.
